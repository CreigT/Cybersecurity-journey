# Cybersecurity Code of Conduct for Students

## Cybersecurity Journey Edition

### Mission

To develop cybersecurity knowledge while maintaining legality, professionalism, integrity, and ethical conduct.

---

## 1. Legality First

Always operate within the law.

Never access systems, data, or networks without explicit authorization.

---

## 2. Integrity and Honesty

Be truthful regarding:

- Skills
- Experience
- Certifications
- Research

Always credit original sources.

---

## 3. Responsible Learning

Practice only in:

- Virtual Machines
- Home Labs
- TryHackMe
- Hack The Box
- VulnHub
- Authorized environments

---

## 4. Professional Conduct

Represent the cybersecurity profession positively.

Maintain professionalism on:

- GitHub
- LinkedIn
- Discord
- Reddit
- X

---

## 5. Confidentiality

Protect:

- Passwords
- API Keys
- Personal Data
- Client Information
- Sensitive Reports

---

## 6. Responsible Disclosure

If vulnerabilities are discovered:

1. Document findings.
2. Notify the owner privately.
3. Allow remediation time.
4. Follow disclosure policies.

---

## 7. Responsible AI Usage

Use AI responsibly.

Verify AI-generated information.

Never use AI to:

- Create malware
- Conduct phishing
- Facilitate illegal activity

---

## 8. Portfolio Ethics

Remove:

- Credentials
- Sensitive information
- Client data
- Internal documents

before publishing.

---

## 9. Continuous Learning

Remain curious.

Remain humble.

Support other learners.

Share knowledge responsibly.

---

## Student Pledge

I commit to ethical cybersecurity learning, responsible disclosure, professional conduct, and continuous improvement.

---

Stay Curious.
Stay Ethical.
Stay Professional.
